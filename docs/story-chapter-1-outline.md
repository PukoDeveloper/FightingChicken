# 第一章故事大綱：星火與勇氣

> 目標：將目前故事模式從「幾段章節對白」擴展為可支撐 50 關的第一大章內容。
> 這份文件先定義劇情方向、角色弧線、關卡節點與每關敘事目的，後續再拆成實際 `StoryScene`、關卡資料與戰鬥事件。

## 核心定位

- **第一章名稱**：星火與勇氣
- **關卡規模**：50 關
- **主要主題**：勇氣不是不害怕，而是承認恐懼後仍選擇前進。
- **情緒曲線**：出發的天真 -> 被迫戰鬥 -> 失去方向 -> 找到同伴 -> 直面真相 -> 主動選擇承擔。
- **玩法敘事方向**：
  - 前 10 關：建立小雞、勇氣與祖父失蹤事件。
  - 11-20 關：幻影與記憶主題，讓玩家懷疑「祖父是否真的想被救」。
  - 21-30 關：混沌勢力浮現，隊伍內部價值衝突加深。
  - 31-40 關：水晶塔與機械守衛，揭露祖父其實曾參與封印系統。
  - 41-50 關：暴風、龍王與虛空核心前哨，第一章以救出祖父但打開更大危機收束。

## 世界觀簡述

星空不是單純的宇宙，而是一片由「記憶、意志、願望」共同構成的航道。強烈的情感會凝結成星塵，也會吸引吞噬情感的異象。

很久以前，祖父火雞與幾位守護者建立了「水晶塔」，用來封住星空深處的虛空裂隙。多年後封印鬆動，混沌利用幻影竊取記憶、用機械守衛重啟塔內協議，試圖把小雞引到核心，讓牠用純粹的勇氣點燃裂隙。

第一章表面目標是「救回祖父」，真正目標是讓小雞理解：祖父不是等待被拯救的老人，而是曾經犯錯、仍努力補救的人；勇氣也不是單純的敵人，而是小雞缺少的另一半性格。

## 主要角色立體化

### 小雞

- **表面形象**：熱血、善良、直覺行動。
- **真正缺口**：害怕自己其實沒有能力救任何人，所以常用衝動掩飾不安。
- **渴望**：救回祖父，證明自己已經長大。
- **恐懼**：祖父失蹤是因為自己太弱；同伴受傷也是自己的錯。
- **第一章轉變**：
  - 1-10 關：以為勇氣等於「不能退縮」。
  - 11-25 關：被幻影逼著看見自己的嫉妒、恐懼與依賴。
  - 26-40 關：學會聽同伴意見，不再把所有責任攬在自己身上。
  - 41-50 關：選擇救祖父，也願意承擔打開虛空線索的後果。
- **對白語氣**：句子短、情緒直接；後期開始能說出「我害怕，但我還是要做」。

### 勇氣

- **表面形象**：粗魯、好戰、愛用考驗包裝關心。
- **真正缺口**：牠是水晶塔外圍守護意志的化身，害怕承認自己也會恐懼。
- **渴望**：找到值得託付星火的人。
- **恐懼**：如果自己放行錯的人，虛空會再次被打開。
- **第一章轉變**：
  - 起初把小雞當入侵者。
  - 中段成為同行者，但仍習慣用命令保護小雞。
  - 後段承認自己不是「勇氣本身」，只是曾經失敗的守門人。
  - 第 50 關前正式把「星火徽記」交給小雞。
- **對白語氣**：外硬內軟，常說「哼」或「別誤會」；越後期越少挑釁，多出提醒與自責。

### 祖父火雞

- **表面形象**：慈祥導師、被囚禁的救援目標。
- **真正缺口**：曾參與水晶塔封印，卻在封印虛空時犧牲了部分同伴記憶。
- **渴望**：讓小雞活在不需要背負過去錯誤的世界。
- **恐懼**：小雞知道真相後會恨自己。
- **第一章功能**：
  - 前期以回憶與斷訊聲音提供情感牽引。
  - 中期被幻影模仿，製造真假祖父的懸念。
  - 後期揭露自己不是單純受害者，而是封印系統的共同建造者。
  - 第 50 關被救出，但坦承虛空核心還未真正關閉。
- **對白語氣**：溫柔、克制，常用比喻；真相揭露後語句變短，帶愧疚。

### 幻影

- **表面形象**：嘲笑、迷惑、竊取記憶的敵人。
- **真正缺口**：幻影本身沒有固定記憶，只能靠吞食他人記憶確認自己存在。
- **渴望**：獲得一段「屬於自己」的回憶。
- **恐懼**：被遺忘，或發現自己只是混沌製造的工具。
- **第一章轉變**：
  - 11-20 關作為主要反派，用祖父幻象折磨小雞。
  - 21-30 關開始透露混沌並不信任它。
  - 35 關後可短暫協助小雞，因為它想知道自己被創造前的真相。
  - 48-50 關可留下伏筆：幻影沒有死，而是帶走一段祖父遺失的記憶。
- **對白語氣**：詩意、斷裂、愛重複別人的話；情緒動搖時會突然說得很像小雞或祖父。

### 混沌

- **表面形象**：傲慢、玩弄人心的幕後主使。
- **真正缺口**：混沌不是單純邪惡，而是星空中所有「不被承認的恐懼」聚合出的意志。
- **渴望**：證明所有勇氣最後都會變成自欺。
- **恐懼**：有人能承認恐懼卻不被它吞沒。
- **第一章功能**：
  - 21-30 關正式登場，誘導小雞把「救祖父」變成執念。
  - 31-40 關利用水晶塔協議，把祖父過去錯誤放大。
  - 41-50 關退居幕後，讓暴風與龍王測試小雞是否有資格靠近虛空核心。
- **對白語氣**：優雅、嘲諷，不急著殺死主角；喜歡把真話說成像謊言。

### 水晶守衛 / 機甲

- **表面形象**：冷冰冰的防衛機器。
- **真正缺口**：它保留了祖父當年輸入的保護指令，卻因資料破損把「保護」解讀成「禁止所有接近」。
- **渴望**：完成最後一條命令。
- **恐懼**：沒有情緒，但它的行為呈現「害怕失職」的程式偏執。
- **第一章轉變**：
  - 31-40 關從敵人變成悲劇性守門者。
  - 被擊敗後不是毀滅，而是把塔內地圖交給小雞。
- **對白語氣**：短句、判定式、重複協議；越後期越出現故障般的人性片段。

### 暴風魔

- **表面形象**：速度快、情緒不穩、像災害一樣的敵人。
- **真正缺口**：它曾是水晶塔的信使，因長年穿越裂隙而被風暴侵蝕。
- **渴望**：停止奔跑，找到能把訊息送達的人。
- **恐懼**：一停下來就會聽見自己失敗時同伴的聲音。
- **第一章功能**：
  - 41-45 關測試小雞能否在混亂中保持判斷。
  - 戰敗後交出「裂隙座標」，也提醒祖父可能隱瞞了更大的代價。
- **對白語氣**：急促、跳躍、句尾常被打斷。

### 龍王

- **表面形象**：壓倒性的火焰 Boss、古老守護者。
- **真正缺口**：它不是混沌手下，而是水晶塔最古老的守門盟友；它認為救一個人不值得冒打開虛空的風險。
- **渴望**：守住大局，即使要被所有人憎恨。
- **恐懼**：再次相信年輕英雄，卻重演過去失敗。
- **第一章功能**：
  - 46-49 關作為價值對立者，不是純反派。
  - 第 49 關後讓路，但要求小雞親口承諾：若祖父已被虛空污染，必須做出選擇。
- **對白語氣**：莊重、古老、少廢話；把每句話都說得像判決。

### 虛空核心

- **表面形象**：黑洞、終局異象。
- **真正缺口**：它不說謊，也不誘惑；它只是把角色心中最空缺的部分放大。
- **第一章功能**：
  - 第 50 關不完全決戰，只是第一次直視。
  - 它讓小雞看見「救出祖父後，祖父仍可能再次離開」。
  - 第一章結尾留下第二章主軸：如何面對被救回來但不再完整的家人。

## 每 10 關的章節鉤子

### 1-10：遇見勇氣

- 小雞從單純尋親者變成被星路承認的挑戰者。
- 勇氣從敵人轉為不情願的同行者。
- 結尾鉤子：祖父的聲音被幻影竊用。

### 11-20：幽靈之謎

- 幻影攻擊小雞最脆弱的記憶。
- 小雞第一次理解「想救祖父」也可能混著自私與恐懼。
- 結尾鉤子：幻影吐出「混沌」之名，並暗示祖父不是單純受害者。

### 21-30：混沌的降臨

- 混沌利用真相製造裂痕，而不是只靠謊言。
- 勇氣承認過去失敗，小雞學會重新信任。
- 結尾鉤子：水晶塔座標出現，祖父與封印系統連上線。

### 31-40：水晶塔試煉

- 機械守衛讓「保護」變成反派力量。
- 祖父的人設從完美導師變成曾犯錯的保護者。
- 結尾鉤子：塔內地圖指出祖父被困在虛空門扉旁。

### 41-50：虛空門扉

- 暴風魔與龍王不是單純敵人，而是不同形式的守護者。
- 小雞學會承認害怕，並在知道代價後仍做選擇。
- 結尾：祖父被救出，但虛空核心看見了小雞的星火；第二章危機開始。

## 對白與角色寫作準則

- **每段對白都要帶有角色需求**：小雞想前進、勇氣想保護、幻影想確認存在、混沌想動搖信念。
- **不要讓反派只解釋設定**：反派說設定時，必須同時刺中主角弱點。
- **同一個事件給不同角色不同解讀**：
  - 小雞：祖父被困，所以要救。
  - 勇氣：祖父被困可能是封印的一部分，所以要謹慎。
  - 混沌：祖父被困證明勇氣終究只會傷害所愛之人。
  - 龍王：祖父被困是代價，不該為一人毀掉封印。
- **角色成長要落在行動上**：
  - 小雞後期會先問同伴意見，再衝。
  - 勇氣後期會說「我也害怕」。
  - 幻影後期會保留一段記憶而不是吞掉它。
  - 祖父後期會道歉，而不是只給教訓。

## 後續落地建議

1. 將目前序章與 1-5 章對白重編為 1-10 關的劇情片段。
2. 新增 `storyLevels` 或類似資料層，讓故事模式支援 50 個節點，而不是一章一個場景檔。
